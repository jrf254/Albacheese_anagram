Instructions:

Select letters from the grid to build words. When a word is completed. Press the enter button to submit the word.


Use the clear button to delete a word from the input


Use the shuffle button to shuffle the letters on the board


Use the "+" button to start a new game, and use the "||" button to end your current game.
